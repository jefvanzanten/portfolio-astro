# Project Card Description Guidelines

Laatste meting: 2026-06-25

Basis: alle projectbeschrijvingen in `src/data/projects`, exclusief `idea-leaderboard.md` en `note-markdown-apps.md` omdat die op dat moment duidelijk langer waren dan de rest.

- Gemiddelde lengte overige kaarten: 211.8 tekens
- Mediaan overige kaarten: 210 tekens
- Praktische richtlijn: mik op 180 tot 220 tekens
- Voorkeur: 1 korte alinea
- Vermijd details die ook al zichtbaar zijn in tags, titel of repository-link

Doel: de kaarten visueel gelijk houden, zodat beschrijvingen niet onverwacht extra hoogte veroorzaken.
